# Asteroids

A classic arcade-style Asteroids game built using Python and the Pygame library. This was created as a guided project through [Boot.dev](https://www.boot.dev).

## Description

In this game, you control a spaceship in a field of asteroids. Your goal is to survive as long as possible by maneuvering and shooting down the asteroids before they collide with your ship.

## Features

- **Smooth Movement:** Physics-based rotation and acceleration.
- **Classic Gameplay:** Large asteroids break into smaller ones when hit.
- **Clean Code:** Built using Object-Oriented Programming (OOP) principles like inheritance and sprite groups.
